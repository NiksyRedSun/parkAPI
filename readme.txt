1) Данный проект создан для контроля парковочных мест в пределах одного дома
2) Все зависимости находятся в файле requierements.txt
3) Перед запуском проекта необходимо установить зависимости.
Создать в директории файл .env и поместить в него информацию для установления связи с базой данных.
Необходимые к созданию переменные окружения можно найти в файле config.py.
После чего необходимо привести базу данных в необходимое состояние, с помощью команды "flask db upgrade head",
прописанной в консоль, находясь в виртуальном окружении проекта.
Для заполнения базы данных начальными значениями можно использовать роутер из файла, fill_db.py.
Для этого нужно перенести его в файл app.py, запустить приложение, и один раз перейти по пути /test
После чего база данных будет иметь квартиры и парковочные места. Можно приступать к её заполнению
